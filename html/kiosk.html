KIOSK

Key Features

  • Tip Selection Options: Users can either enter a custom tip amount or choose a randomly generated tip below 25% of the total bill.
  • Review & Rating System: Customers can rate their experience (1-5 stars) and leave a short text review.
  • Receipt Generation & Printing: The system calculates the final bill, displays it on-screen, and formats the receipt for easy printing.


    • Technology Used

    Programming Language: C++


    Potential Enhancements:

  • Menu System: Allow customers to select and order items from a predefined menu.
  • Payment Integration: Simulate card/cash payments for a more realistic experience.
  • Database Storage: Store order history and reviews in a file or database.
  • Graphical Interface: Upgrade to a GUI-based application using frameworks like Qt or SFML.
  • This project is an ideal foundation for a real-world self-service kiosk system, with room for further development and customization.
